# WAIQ Support

Support and issue tracking for **WAIQ**, the AI radio host for Apple Music. WAIQ plays your music and, before each track, speaks a genuinely interesting, true line about the song — old-school radio, hosted by an AI.

> This repository is for support only. It does not contain the app's source code.

## Getting help

- **Found a bug?** [Open a bug report](../../issues/new?template=bug_report.yml)
- **Have an idea?** [Request a feature](../../issues/new?template=feature_request.yml)
- **No GitHub account?** Email **johndifini@gmail.com** with "WAIQ" in the subject line.

When reporting a problem, please include your iPhone model, iOS version, and what you were doing when it happened.

## Frequently asked questions

**Why don't I hear any music?**
WAIQ needs an active **Apple Music subscription** to play your library, playlists, and stations. Demo Mode works without one — it runs the full DJ experience over a simulated playlist.

**Why is the DJ silent?**
The spoken commentary is generated by AI and requires an **Anthropic API key**, entered once in the app's Settings. An optional ElevenLabs key upgrades the voice; without it, WAIQ uses the built-in iOS voice.

**Does WAIQ keep playing when my phone is locked?**
Yes — the show continues in the background with the screen off.

**Privacy**
WAIQ has no accounts and collects nothing. Your API keys are stored in your device's Keychain and never leave it. Song metadata (title, artist, album) is sent to the AI service only to write each on-air line.
